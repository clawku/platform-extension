# Screenshots & Images Guide

## Required Screenshots (1280x800 px)

Take these screenshots for the Chrome Web Store listing:

### Screenshot 1: Extension Popup - Paired State
1. Open Chrome with the extension installed and paired
2. Click the extension icon in the toolbar
3. Capture the popup showing:
   - Connected status indicator (green)
   - Persona name
   - "Disconnect" button

### Screenshot 2: Extension Popup - Pairing Screen
1. Install the extension (or disconnect if already paired)
2. Click the extension icon
3. Capture the popup showing:
   - 6-digit code input field
   - "Pair with Clawku" button
   - API URL field (optional)

### Screenshot 3: AI Browser Navigation
1. Open WhatsApp Web (or Telegram Web)
2. Send a command to your AI: "Go to google.com"
3. Screenshot should show:
   - Chat window with the command
   - Browser navigating to Google

### Screenshot 4: AI Taking Screenshot
1. Navigate to any interesting webpage
2. Ask your AI: "Take a screenshot of this page"
3. Screenshot should show:
   - The webpage
   - AI response with screenshot (or WhatsApp showing the screenshot)

### Screenshot 5: AI Form Interaction
1. Navigate to a page with a form (like Google search)
2. Ask your AI: "Search for 'Clawku AI'"
3. Screenshot should show the AI completing the action

## Tools for Creating Screenshots

### macOS
```bash
# Full screen: Cmd + Shift + 3
# Selection: Cmd + Shift + 4
# Window: Cmd + Shift + 4, then Space, click window

# Resize to exact dimensions:
sips -z 800 1280 screenshot.png
```

### Chrome Extension for Screenshots
- GoFullPage (for full page screenshots)
- Awesome Screenshot

### Design Tools
- Figma (free)
- Canva (free)

## Screenshot Tips

1. **Use a clean browser profile** - No personal bookmarks or extensions visible
2. **High resolution display** - Take at native resolution, then resize
3. **Consistent theme** - Use light mode for all screenshots
4. **Hide sensitive data** - Blur or crop any personal information
5. **Annotate if needed** - Add arrows or highlights to show key features
